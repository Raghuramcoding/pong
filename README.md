# Pong Game

## Instructions on How to Play
1. Open your web browser.
2. Navigate to the directory where the Pong game files are located.
3. Open the `index.html` file in your browser.
4. Use the keyboard to control the paddles:
   - Player 1: Press 'W' to move up and 'S' to move down.
   - Player 2: Press the 'Up Arrow' to move up and 'Down Arrow' to move down.
5. The objective of the game is to score points by getting the ball past your opponent's paddle.

Enjoy playing Pong!
